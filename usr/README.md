# NexCRM

A modern, responsive Customer Relationship Management (CRM) application built with Flutter. NexCRM helps sales teams manage their contacts, track deal pipelines, and visualize their progress with powerful dashboards.

## Features

- **Dashboard:** At-a-glance metrics including pipeline value, active deals, total won, and win rate. Features an interactive pipeline growth chart.
- **Pipeline View:** A Kanban-style board to track deals across different stages (Leads, Qualified, Proposal, Negotiation, Closed Won, Closed Lost).
- **Contacts Directory:** A searchable list of all clients and prospects, displaying key contact info and company association.
- **Responsive Design:** Adapts smoothly between a bottom navigation bar on mobile devices and a side navigation rail on tablets and desktops.

## Tech Stack

- **Framework:** Flutter 
- **Typography:** Google Fonts
- **Charts:** fl_chart for dashboard data visualization
- **Formatting:** intl for currency and date formatting

## Setup and Run Instructions

1. Ensure you have Flutter installed (`flutter doctor`).
2. Run `flutter pub get` to fetch dependencies.
3. Start the application:
   ```bash
   flutter run
   ```
